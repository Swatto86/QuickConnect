üìñ Quick Start Guide

  1. Download and install QuickConnect from the releases page
  2. Launch and enter your domain credentials (format: DOMAIN\username or user@domain.com)
  3. Add hosts manually or click "Scan Domain" to auto-discover servers from Active Directory
  4. Click any host to instantly connect via RDP with your saved credentials
  5. Use the system tray for quick access to recent connections and settings
